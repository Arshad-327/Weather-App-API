# 🌤️ SkyCheck — Weather App

A real-time weather lookup app built using HTML, CSS, JavaScript, and the OpenWeatherMap API.

---

## 📌 About This Project

This is my first project as a B.Tech IT student learning web development hands-on.

No frameworks. No fancy setup. Just HTML, CSS, JavaScript, and one free API.

---

## 🚀 Features

* Real-time weather data for any city
* Live temperature, humidity, wind speed display
* Sunrise & sunset times
* Min / Max temperature with visual bar
* Weather condition with matching emoji
* Error handling for invalid city names
* Clean dark sky UI with animations

---

## 🔧 Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeatherMap API
